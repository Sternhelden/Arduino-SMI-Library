smi
=========

smi is a library for accessing Davicom products through SMI.


Version
----

1.0


Initialize
--------------

Using SMI to configure the object with clock pin  and data pin
```sh
SMI smi(5,4);

```
 - smi => object
 - 5 => clock pin
 - 4 => data pin


Functions
--------------
 - read();
 - write();
